## Run Locally

Clone the project

```bash
  git clone https://github.com/DidoeS14/work-tracker-frontend.git
```

Go to the project directory

```bash
  cd work-tracker-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run build
```
